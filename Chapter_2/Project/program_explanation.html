Greeting Program Using JavaScript

Here's a simple example of an HTML and JavaScript code snippet that creates a page with an input field for the user's name. When the user types their name and clicks the "Submit" button, it will display the greeting "Hello, [name]".

When you open this HTML page in a browser, it will allow you to enter your name, for example Valery, and upon submission, it will display the greeting message Hello, Valery! on the same page. 


    <!DOCTYPE html>
    <html lang="en">
    <head>
      <title>Simple Greeting Page</title>
      <meta charset="utf-8">
    </head>
    <body>
      <h1>Enter your name:</h1>
      <form id="nameForm">
        <input type="text" id="nameInput" placeholder="Your Name">
        <input type="submit" value="Submit">
      </form>

      <div id="greeting" style="display: none;">
        <h2>Greeting:</h2>
        <p id="greetingText"></p>
      </div>

      <script>
        document.getElementById("nameForm").addEventListener("submit", function (event) {
            event.preventDefault(); // Prevent form submission
            var name = document.getElementById("nameInput").value;
            var greetingText = "Hello, " + name + "!";
            document.getElementById("greetingText").textContent = greetingText;
            document.getElementById("greeting").style.display = "block";
        });
      </script>
    </body>
    </html>

In this code:
We have an input field with the placeholder "Your Name" and a "Submit" button inside a form element. When the user submits the form, the JavaScript code prevents the default form submission behavior using event.preventDefault(). It then retrieves the value entered in the input field, constructs the greeting message, and displays it in a <p> element with the id "greetingText." The "greeting" <div> element is initially hidden with style="display: none;" and is set to be displayed (style.display = "block";) after the user submits the form.

This program contains embedded Cascading Style Sheets (CSS) code. It's just a simplified example of how JavaScript can be used to interact with the HTML page.
